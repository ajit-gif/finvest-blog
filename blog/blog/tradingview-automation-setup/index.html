TradingView Automation Setup Guide

Learn how to connect TradingView strategies with FinvestAlgo and automate your trades using webhooks and broker APIs.

🟢 Step 1: Connect Your Broker

What to do:

  • Login to FinvestAlgo
  • Go to Broker Connection / Demat Section
  • Click on Add Account/Broker
  • Select your broker
  • Enter API credentials
  • Click Connect
💡 This step connects your trading account to the automation system. You only need to do this once.

🟢 Step 2: Create Strategy Bridge

What to do:

  • Go to Bridge Section
  • Click Add Strategy
  • Enter Strategy Name
  • Create an 8 character Strategy Key
  • Click Create
🎯 This bridge connects TradingView alerts with broker execution.

🟢 Step 3: Configure Trade Execution

Set the following:

  • Exchange → NSE / NFO
  • Symbol → Stock / Index
  • Order Type → Market / Limit
  • Quantity → Lot Size

For Options Trading:

  • Select CE / PE
  • Choose Strike
  • Select Expiry
💡 You can automate Entry, Exit and Reverse Positions.

🟢 Step 4: Copy Webhook URL

  • Open your created strategy
  • Click Webhook / Copy Button
  • Copy the webhook URL
⚠️ This is the MOST IMPORTANT STEP. This URL connects TradingView to your broker.

🟢 Step 5: Setup TradingView Strategy

  • Open TradingView
  • Click on Indicators
  • Click Create Alert

🟢 Step 6: Configure Alert

Inside Alert Settings:

Condition: Select your strategy

Trigger: Once Per Bar Close

{{strategy.order.action}}

🟢 Step 7: Paste Webhook URL

  • Go to Notifications Tab
  • Enable Webhook URL
  • Paste your webhook URL

🟢 Step 8: Activate Automation 🚀

Click Create Alert

🎯 Done. Your system will now:
  • Detect signals
  • Send webhook
  • Execute trades automatically
