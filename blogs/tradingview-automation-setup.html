TradingView Automation Setup Guide

Complete step-by-step guide to automate your trading strategies using TradingView alerts, broker APIs and FinvestAlgo execution engine.

01

Connect Your Broker

Connect your broker account with FinvestAlgo using broker APIs.

  • Login to FinvestAlgo
  • Go to Broker Connection Section
  • Click Add Broker
  • Select Broker
  • Enter API Credentials
  • Click Connect
02

Create Strategy Bridge

Create a bridge to connect TradingView alerts with broker execution.

  • Open Bridge Section
  • Click Add Strategy
  • Enter Strategy Name
  • Create Strategy Key
  • Click Create
03

Configure Trade Execution

Configure exchange, quantity and execution settings.

  • Select Exchange
  • Select Symbol
  • Choose Order Type
  • Set Quantity
  • Configure Entry & Exit
04

Configure Options Trading

Setup options trading configuration for automation.

  • Select CE / PE
  • Choose Strike Price
  • Select Expiry
  • Configure Quantity
05

Copy Webhook URL

Copy webhook URL to connect TradingView with FinvestAlgo.

  • Open Strategy
  • Click Webhook Button
  • Copy Webhook URL
06

Setup TradingView Strategy

Create TradingView alert and connect strategy conditions.

  • Open TradingView
  • Click Indicators
  • Create Alert
  • Select Strategy
07

Paste Webhook URL

Paste webhook URL inside TradingView notifications tab.

  • Open Notifications Tab
  • Enable Webhook URL
  • Paste Webhook
  • Save Settings
08

Activate Automation

Activate automation and execute trades automatically.

  • Click Create Alert
  • Signals will trigger
  • Webhook will execute
  • Trades execute automatically
