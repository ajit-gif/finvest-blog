TradingView Automation Setup Guide

Complete step-by-step guide to automate your trading strategies using TradingView alerts, broker APIs and FinvestAlgo execution engine.

🟢 Step 1: Connect Your Broker

Connect your broker account with FinvestAlgo using secure broker APIs.

  • Login to FinvestAlgo
  • Go to Broker Connection / Demat Section
  • Click Add Account/Broker
  • Select your broker
  • Enter API credentials
  • Click Connect

🟢 Step 2: Create Strategy Bridge

Create a bridge that connects TradingView alerts with broker execution.

  • Go to Bridge Section
  • Click Add Strategy
  • Enter Strategy Name
  • Create Strategy Key
  • Click Create

🟢 Step 3: Configure Trade Execution

Configure exchange, quantity, order type and trading execution settings.

  • Select Exchange
  • Select Symbol
  • Choose Order Type
  • Set Quantity
  • Configure Entry & Exit

🟢 Step 4: Configure Options Trading

Setup options trading automation with strike and expiry configuration.

  • Select CE / PE
  • Choose Strike Price
  • Select Expiry
  • Configure Quantity

🟢 Step 5: Copy Webhook URL

Copy your webhook URL to connect TradingView alerts with FinvestAlgo.

  • Open Strategy
  • Click Webhook Button
  • Copy Webhook URL

🟢 Step 6: Setup TradingView Strategy

Create TradingView alert and configure strategy conditions.

  • Open TradingView
  • Click Indicators
  • Create Alert
  • Select Strategy
  • Choose Once Per Bar Close

🟢 Step 7: Paste Webhook URL

Paste webhook URL inside TradingView notifications tab.

  • Open Notifications Tab
  • Enable Webhook URL
  • Paste FinvestAlgo Webhook
  • Save Alert Settings

🟢 Step 8: Activate Automation 🚀

Activate automation and execute trades automatically.

  • Click Create Alert
  • TradingView detects signals
  • Webhook sends execution request
  • Broker executes trades automatically
