TradingView Automation Setup Guide

Complete step-by-step setup guide for TradingView automation using FinvestAlgo.

🟢 Step 1: Connect Your Broker

  • Login to FinvestAlgo
  • Go to Broker Connection / Demat Section
  • Click Add Account/Broker
  • Select your broker
  • Enter API credentials
  • Click Connect

🟢 Step 2: Create Strategy Bridge

  • Go to Bridge Section
  • Click Add Strategy
  • Enter Strategy Name
  • Create Strategy Key
  • Click Create

🟢 Step 3: Configure Trade Execution

  • Select Exchange
  • Select Symbol
  • Choose Order Type
  • Set Quantity

🟢 Step 4: Copy Webhook URL

  • Open Strategy
  • Copy Webhook URL

🟢 Step 5: Setup TradingView Alert

  • Open TradingView
  • Create Alert
  • Paste Webhook
  • Click Create Alert
